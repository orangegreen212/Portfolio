Welcome

Welcome to my portfolio showcasing various approaches to time series analysis and forecasting. In this project, I delve into the practical application and comparison of three distinct yet powerful methods: the classic **SARIMA** model, Facebook's renowned **Prophet** library, and the versatile **XGBoost** gradient boosting algorithm adapted for time series problems.

Each dedicated section provides an overview of the method, presents key Python code snippets, visualizes the forecasting results on a real dataset, and evaluates model accuracy using standard metrics.

Methods Explored:

  • SARIMA (Seasonal Autoregressive Integrated Moving Average)
  • Prophet (Facebook Prophet)
  • XGBoost (Application for Time Series)
